npm install bcrypt :

bcrypt est une bibliothèque de cryptographie conçue pour sécuriser les informations sensibles, particulièrement utile pour le hashage des mots de passe avant leur stockage dans une base de données. Voici quelques-unes des fonctionnalités clés et raisons d'utiliser bcrypt :

Hashage sécurisé des mots de passe : Plutôt que de stocker les mots de passe en clair, ce qui serait extrêmement risqué en cas de fuite de données, bcrypt permet de les transformer en une chaîne complexe et irréversible. Ainsi, même si les données sont compromises, les mots de passe originaux restent protégés.
Salage automatique : Le "salage" est une technique qui ajoute des données aléatoires au mot de passe avant de le hasher. Cela empêche les attaques utilisant des tables de hachage précalculées (rainbow tables) pour déchiffrer les mots de passe. bcrypt intègre automatiquement un sel unique à chaque mot de passe avant de le hasher, renforçant ainsi la sécurité.
